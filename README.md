Bindings-Libstemmer
===

low level binding for libstemmer.

## Requirement

* libstemmer_c

### for Ubuntu

```bash
$ sudo apt-get install libstemmer-dev
```

### for OS X

```bash
$ brew tap cosmo0920/tokenizers
$ brew install cosmo0920/tokenizers/libstemmer
```

## Install

```bash
$ cabal install bindings-libstemmer
```
